Manage FTP accounts in the GameAP

## Installation

### Via Composer

Install ftp module:
```bash
composer require gameap/ftp-module "^1.1"
```

Update migrations:
```bash
php artisan module:migrate Ftp
```

### Copying

Copy files to `/path/to/gameap/modules/Ftp/`

Update migrations:
```bash
php artisan module:migrate Ftp
```
